How Is Testing Related to Single Statement Bugs?

Jan 2021 - Apr 2021

  • Observed weak-to-moderate correlation (r = 0.40) between test coverage and single-statement bugs in open-source Java projects.
  • Suggested test coverage improvements to reduce bug prevalence.
